Overview

This project is an AI-powered recruitment prediction system that helps predict whether a candidate will be hired or not based on their profile. It combines Machine Learning models with Explainable AI (XAI) techniques to improve transparency and reduce bias in hiring decisions.

✨ Features
📂 Upload dataset (CSV / Excel)
🤖 Train multiple ML models:
Logistic Regression
Multi-Layer Perceptron (MLP)
XGBoost (optional)
📊 Model evaluation metrics:
Accuracy
Precision
Recall
F1 Score
Confusion Matrix
ROC Curve
🔍 Candidate prediction interface
🧠 Explainable AI:
SHAP (global & local explanations)
LIME (local explanations)
📈 Interactive dashboard using Streamlit
🛠️ Tech Stack
Python
Streamlit
Scikit-learn
TensorFlow (Keras)
XGBoost
SHAP & LIME
Pandas, NumPy, Matplotlib
📁 Project Structure
├── app.py                # Main Streamlit application
├── dataset/              # Dataset files
├── models/               # Saved models (optional)
├── README.md             # Project documentation
└── requirements.txt      # Dependencies
